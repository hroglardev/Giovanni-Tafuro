# Giovanni Tafuro

El proyecto es un proyecto introductorio a HTML para usar etiquetas semanticas basicas.

Las etiquetas utilizadas fueron:

<ul>
  <li>header</li>
  <li>main</li>
  <li>footer</li>
  <li>section</li>
  <li>figure</li>
  <li>img</li>
  <li>figcaption</li>
  <li>hr</li>
  <li>h1</li>
  <li>h2</li>
  <li>h3</li>
  <li>p</li>
</ul>
